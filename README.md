# 《信息几何方法》——书籍摘要与目录

本书是信息几何领域的经典著作，由 Amari Shun-ichi 和 Nagaoka Hiroshi 合著，英文版由美国数学学会（AMS）出版（2000年）。本书系统介绍了信息几何的数学基础及其在统计学、信息论、系统科学、量子信息等领域的应用。

---

## 📖 书籍简介

信息几何起源于对概率分布族所固有的微分几何结构的研究。例如，正态分布族可视为一个二维流形，其上自然诱导出 Fisher 度量，并且该流形具有常负曲率。更一般地，概率分布族不仅具有黎曼度量，还引出对偶仿射连接这一新概念，这是经典微分几何中未曾深入探讨的结构。

本书首先用三章篇幅简明介绍微分几何与对偶连接理论，然后逐步展示该理论在统计推断、时间序列与线性系统、多终端信息论、量子系统等领域的应用。书中还涵盖了凸分析、神经流形、李群变换模型等前沿交叉内容。

本书适合对微分几何、统计学、信息论、控制论、量子物理等感兴趣的读者，既可作为入门教材，也可作为研究参考书。
